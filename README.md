# Gestión de Clientes - Documentación

Aplicación para gestión de clientes con integración a bases de datos SQL Server y notificaciones vía WhatsApp.

**Características principales:**
- ✅ **CRUD de Clientes**  
- ✅ **Cifrado de Credenciales**  
- ✅ **Envío masivo de mensajes por WhatsApp**
- ✅ **Auditoría de Eventos** 
- ✅ **Manejo de Sesiones Seguras**
- ✅ **Integración con WhatsApp**
