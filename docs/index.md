# Documentación Técnica - Sistema de Gestión de Clientes

*Última actualización: 11 de abril de 2025*

## Introducción

Bienvenido a la documentación técnica del Sistema de Gestión de Clientes. Esta aplicación ha sido desarrollada para administrar de forma eficiente la información de clientes, con integración a bases de datos SQL Server y capacidades de comunicación a través de WhatsApp.

Esta documentación está dirigida a desarrolladores, administradores de sistemas y personal técnico responsable de la implementación, mantenimiento y mejora del sistema.

## Características Principales

- ✅ **CRUD de Clientes**: Gestión completa de información de clientes
- ✅ **Cifrado de Credenciales**: Almacenamiento seguro de información sensible
- ✅ **Envío masivo de mensajes por WhatsApp**: Comunicación automatizada con clientes
- ✅ **Auditoría de Eventos**: Registro detallado de todas las operaciones
- ✅ **Manejo de Sesiones Seguras**: Control de acceso robusto
- ✅ **Integración con WhatsApp**: API de comunicación bidireccional
