# Requisitos del Sistema

## Software
- Windows 10/11.
- Python 3.8 o superior.
- Microsoft ODBC Driver for SQL Server.
- SQL Server 2016+ accesible (puede ser instancia local o remota).

## Python (pip)
Instalar librerías requeridas:
```
pip install pyodbc tkinter cryptography keyring matplotlib tkcalendar requests win10toast pillow
```

## Hardware sugerido
- CPU doble núcleo o superior.
- RAM 4GB (8GB recomendado).
- Almacenamiento 1GB libre.
- Resolución mínima 1366x768.

## Previos
- Crear/usar base de datos con acceso desde la máquina cliente.
- Tener un token de WhatsApp Business válido (opcional si no se usa mensajería).

Para instalar, ver la [Guía de Instalación](./pasos_instalacion.md).
