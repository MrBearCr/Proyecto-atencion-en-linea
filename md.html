Gestión de Clientes - Corporativo

Aplicación para gestionar registros de clientes y enviar mensajes a través de WhatsApp.

Introducción Instalación Configuración Uso Referencia de API Errores y Soluciones

Introducción

Esta aplicación permite gestionar registros de clientes y enviar mensajes a través de WhatsApp. Utiliza una interfaz gráfica amigable y ofrece funcionalidades para buscar, guardar, actualizar y eliminar registros de clientes, así como enviar mensajes masivos.

Instalación

Para instalar la aplicación, sigue estos pasos:

  1. Clona el repositorio: 
    git clone https://github.com/tu-repo/db-client-app.git
cd db-client-app
  2. Instala las dependencias: 
    pip install -r requirements.txt
  3. Ejecuta la aplicación: 
    python app.py

Configuración

La aplicación requiere una configuración inicial para conectarse a la base de datos y a la API de WhatsApp.

Archivo de Configuración

El archivo de configuración se encuentra en db_config.ini. Ejemplo de contenido:

[Database]
server = your_server_address
database = your_database_name
user = your_username
password = your_encrypted_password

Credenciales

Las credenciales sensibles como contraseñas y tokens de API se manejan mediante keyring. Para almacenar estas credenciales, utiliza los siguientes comandos:

keyring set DBClientApp encryption_key your_encryption_key
keyring set DBClientApp temp_pass your_encrypted_temp_password
keyring set DBClientApp whatsapp_token your_encrypted_whatsapp_token

Uso

Interfaz de Usuario

La aplicación tiene una interfaz gráfica amigable con el usuario. Aquí están algunas de sus funcionalidades:

  • Registros: Permite buscar, guardar, actualizar y eliminar registros de clientes.
  • Mensajería: Permite enviar mensajes masivos a través de WhatsApp.
  • Configuración: Permite configurar la conexión a la base de datos y a la API de WhatsApp.

Referencia de API

Conexión a la Base de Datos

Método connect

Conecta a la base de datos utilizando las credenciales proporcionadas.

Parámetros
  • server: Dirección del servidor.
  • database: Nombre de la base de datos.
  • user: Nombre de usuario.
  • password: Contraseña.
Retorno
  • True si la conexión es exitosa.

Errores y Soluciones

Error de Conexión a la Base de Datos

Mensaje:

Error de conexión a la base de datos

Solución:

  • Verifica que las credenciales de conexión a la base de datos sean correctas y que el servidor esté disponible.
